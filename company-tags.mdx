## What are company tags?

At the top of every solution file, a comment block lists all the companies known to have asked that problem in real coding interviews. This lets you go straight to the problems most relevant for your specific target company.

Here is the exact format as it appears in the source files:

```cpp
/*
    MY YOUTUBE VIDEO ON THIS Qn : https://www.youtube.com/watch?v=TCaBnVIllrQ
    Company Tags                : Zoho, Flipkart, Morgan Stanley, Accolite, Amazon,
                                  Microsoft, FactSet, Hike, Adobe, Google, Wipro,
                                  SAP Labs, CarWale
    Leetcode Link               : https://leetcode.com/problems/two-sum/
*/
```

Another example from a graph problem:

```cpp
/*
    Company Tags : Paytm, Amazon, Microsoft, Samsung, Snapdeal, Citrix, D-E-Shaw,
                   Ola Cabs, Visa, Linkedin, Opera, Streamoid Technologies, Informatica
*/
```

## How to use company tags for targeted prep

If you know which company you're interviewing at, you can search across the entire repository for all problems tagged with that company.

<Tip>
  Use `git grep` to instantly find every solution file tagged with a specific company:

  ```bash
  git grep "Company Tags.*Amazon" --include="*.cpp"
  ```

  Replace `Amazon` with any company name. This works for any company in the list below. To match partial names, use a substring:

  ```bash
  git grep "Company Tags.*Microsoft" --include="*.cpp"
  ```
</Tip>

Once you have the list of matching files, open each one, attempt the problem on LeetCode, and use the linked video if you need a walkthrough.

<Note>
  Frequency tags are being added to solutions from a certain point onwards. When a frequency value is present in the file header, it indicates how often that problem has appeared in real interviews — useful for deciding what to prioritize within a company's problem set.
</Note>
