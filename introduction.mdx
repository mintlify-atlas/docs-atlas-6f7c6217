Interview DS Algo is a one-stop repository for coding interview preparation, built and maintained by [codestorywithMIK](https://www.youtube.com/channel/UCaw58edcO3ZqMw76Bvs0kGQ). Every solution is written in C++ and comes with company tags, complexity analysis, and a video explanation — all free.

## What's inside

- **900+ problems** covering the full spectrum of coding interview topics
- **20+ topic categories** including Arrays, DP, Graph, Tree, Trie, and more
- **Company tags on every solution** — see which companies have asked each problem
- **YouTube video explanations** in Hinglish for every question
- **Multiple approaches** per problem: brute force through optimal

## Track record

This repository directly helped crack interviews at **Microsoft**, **Tower Research**, and **Tokopedia**.
