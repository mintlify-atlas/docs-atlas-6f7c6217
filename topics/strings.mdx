## Technique Patterns

### Reverse Words — Two Pointer Approach

Reverse the entire string, then reverse each word individually. The `stringstream` approach is cleaner; the two-pointer approach works in O(1) space.

```cpp
// T.C : O(n)   S.C : O(1)
// Company Tags: Amazon, Microsoft, Adobe, Goldman Sachs, Samsung
string reverseWords(string s) {
    // Step 1: reverse the whole string
    reverse(s.begin(), s.end());

    int n = s.length();
    int l = 0, r = 0, i = 0;

    while (i < n) {
        // collect non-space characters
        while (i < n && s[i] != ' ')
            s[r++] = s[i++];

        if (l < r) {
            // reverse the current word
            reverse(s.begin() + l, s.begin() + r);
            s[r] = ' ';
            r++;
            l = r;
        }
        i++;
    }

    return s.substr(0, r - 1);
}
```

### KMP LPS Array — Shortest Palindrome

Compute the Longest Proper Prefix which is also Suffix (LPS) array over `s + "#" + reverse(s)` to find the longest palindromic prefix in O(n).

```cpp
// T.C : O(n)   S.C : O(n)
// Company Tags: Google
void computeLPS(string pattern, vector<int>& lps) {
    int len = 0, i = 1;
    lps[0] = 0;

    while (i < (int)pattern.size()) {
        if (pattern[i] == pattern[len]) {
            lps[i++] = ++len;
        } else {
            if (len != 0)
                len = lps[len - 1];
            else
                lps[i++] = 0;
        }
    }
}

string shortestPalindrome(string s) {
    string rev  = s;
    reverse(begin(rev), end(rev));
    string temp = s + "-" + rev;

    vector<int> lps(temp.size(), 0);
    computeLPS(temp, lps);

    int longestPrefix = lps[temp.size() - 1];
    return rev.substr(0, s.size() - longestPrefix) + s;
}
```

## Tips for Interviews

<Tip>
When a problem involves finding a substring that satisfies a frequency constraint, think sliding window with a character frequency map first. Expand the right pointer, then shrink the left pointer to restore validity.
</Tip>

<Tip>
KMP's LPS (failure function) array has uses beyond pattern matching: it can detect repeated substrings, find the shortest palindrome prefix, and solve "minimum time to revert word" problems. Learn to build the LPS array on its own, not just inside the full KMP search.
</Tip>

<Tip>
In-place string reversal problems (Reverse Words, Reverse Words II) usually follow a two-step recipe: reverse the whole string, then reverse each segment. Recognize this template and code it directly.
</Tip>

<Tip>
For parentheses validity problems (Minimum Remove to Make Valid Parentheses, Minimum Add to Make Parentheses Valid), a stack that tracks unmatched indices is usually cleaner and more adaptable than counter-based approaches.
</Tip>

<Tip>
`std::stringstream` is a powerful C++ tool for tokenizing strings on whitespace. It is worth knowing for interview coding, but be ready to implement the two-pointer version as well since interviewers sometimes restrict library use.
</Tip>
